LAB-7.2 - HTML Form Validation (No RegExp) - Connected Pages
-----------------------------------------------------------

This version updates each page so that when a user clicks 'Submit' and validation succeeds,
the browser automatically opens the next page in the sequence.

Sequence:
page1_name.html -> page2_email.html -> page3_checkboxes.html -> page4_date.html ->
page5_multicheck.html -> page6_select.html -> page7_user_pic.html -> page8_assignment.html

How to use:
1. Download and unzip the folder.
2. Open index.html in Chrome (from your file manager or GitHub Pages).
3. Each page will validate; on success it navigates to the next page.

Notes:
- Navigation happens only when validation passes (Option B requested).
- No RegExp used anywhere.
- page8 shows a success message when all checks pass.
